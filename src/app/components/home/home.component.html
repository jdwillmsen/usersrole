Users Role

Welcome to the Users Role project!

This is an Angular Frontend Application with role base users. It uses firebase for the user authentication and it's assisted by Firebase functions with an Express server for the role authorization.

There are 3 primary routes once authenticated:

  • /home
    • This is the home link and is currently being displayed.
    • This page is not restricted to any logged in user.
  • /profile
    • This is a profile page for viewing information about the logged in user.
    • This page is restricted to users with the user role.
  • /users
    • This is a page for viewing all the users associated with the application.
    • This page is restricted to users with the manager and admin roles.
